# 后端接口固定的内容
指定云厂商、指定云资源类型、指定操作类型、指定接口下的参数：
字段分组、字段顺序、字段参数名、字段中文名、是否必填、字段类型、包括枚举接口（如果有），比如select的选项是枚举接口返回，则平台默认规范是：接口获取（有查看按钮，点击后真的调用一次这个接口，并将返回的json数据展示出来）；如果select的选项是前端写死的，则平台默认规范是：固定选项，点击查看按钮直接展示选项的名称和传递的参数；

# 部门可以配置的内容
每个字段是否可见，默认可见
如果字段是必填的，部门配置为不可见，则部门配置内容只能是默认值，且为必填或者必选项
各个字段类型可能还有区别：
- select类型：可以配置默认选项；若是接口返回的选项，部门可以配置选项的名称和传递的参数，以及是否级联（已实现不用改）；若是前端固定选项，则只能选择对固定选项做删减；
- string类型：可以配置默认值，设置正则校验（可选）
- number类型：可以配置默认值，设置上下限（可选）
- boolean类型：可以配置默认值
- textarea类型：可以配置默认值
